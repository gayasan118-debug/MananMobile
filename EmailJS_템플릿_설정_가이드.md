# EmailJS 템플릿 설정 가이드 (데이터가 안 올 때 해결법)

고객 정보(이름, 전화번호 등)가 이메일에 내용 없이 양식만 오는 경우, **EmailJS 웹사이트**에서 템플릿 설정을 수정해야 합니다.

현재 웹사이트 코드는 `name`, `phone`, `type`, `message` 라는 이름표를 붙여서 데이터를 보내고 있습니다. 
EmailJS 템플릿에서도 이 이름표(`{{ }}`)를 정확히 써줘야 내용이 보입니다.

---

### 설정 방법

1. **[EmailJS 대시보드](https://dashboard.emailjs.com/)**에 로그인합니다.
2. 왼쪽 메뉴에서 **Email Templates**를 클릭합니다.
3. 목록에서 사용 중인 템플릿 ID (**template_le70gei**)를 찾아 클릭합니다.
4. **Design** 또는 **Content** 탭의 내용 입력창에 아래 내용을 그대로 복사해서 붙여넣으세요. (기존 내용을 지우고 넣으셔도 됩니다.)

#### [복사할 내용]
```text
새로운 관심고객 등록이 접수되었습니다.

---------------------------------------------------
■ 고객명 : {{name}}
■ 연락처 : {{phone}}
■ 선호타입 : {{type}}
---------------------------------------------------

■ 문의내용 :
{{message}}
```

5. 상단의 **Save** 버튼을 눌러 저장합니다.
6. 이제 사이트에서 다시 등록을 시도하면 정보가 정상적으로 들어옵니다.

---

### 주의사항
- `{{name}}` 처럼 중괄호 두 개를 꼭 사용해야 합니다.
- 대소문자를 구분하므로 `{{Name}}` 대신 `{{name}}` (소문자)을 사용하세요.
